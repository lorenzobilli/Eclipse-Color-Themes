This is a complete and adapted port of the Eclipse dark theme of Luna/Mars/Neon versions era.

<img src="./example.png">

All colors and font styles have been ported and adapted to be as close as possible to the original Java dark theme for the following programming languages:

- C

- C++

- C#

- Java

- Python

- JavaScript/TypeScript

- HTML (with Angular special attributes)

- CSS/SCSS

- ... with more to come!

Check out the companion Eclipse Classic Light theme in the Marketplace: https://marketplace.visualstudio.com/items?itemName=LorenzoBilli.eclipse-classic-light

Check out also the dedicated GitHub repository: https://github.com/lorenzobilli/Eclipse-color-theme
